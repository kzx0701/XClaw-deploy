# XClaw 功能缺失清单

> 最后更新：2025-06-12
> 
> 本文档记录 XClaw 当前版本（v1.3.0）缺失的功能点，用于后续迭代规划。

---

## 一、已实现功能

| 模块 | 已完成功能 |
|------|-----------|
| 项目管理 | 导入、选择、删除、配置编辑（打包命令/产物目录） |
| 环境管理 | CRUD、配置表单、目录检测、启用/禁用 |
| 服务器管理 | CRUD、连接测试、密码/私钥认证、搜索 |
| 执行功能 | 本地打包、SSH 部署、一键部署、前置检查、产物智能检测 |
| UI/UX | 搜索、空状态引导、响应式布局 |
| 数据管理 | localStorage 持久化、任务历史（50条） |

| 描述 | 部署时传递自定义环境变量（Key-Value 键值对编辑器） |

## 六、关键发现

1. **TaskHistoryPanel 组件未集成** — 已实现但未在项目详情页使用，成本最低的补全点
2. **暗色模式空壳** — CSS 变量已定义但值与亮色相同
3. **AI 推荐功能半成品** — 类型和后端命令已定义，前端 UI 未使用
4. **设置页面空壳** — 仅有标题，无实际内容
5. **数据安全风险** — 密码明文存储，无备份机制
